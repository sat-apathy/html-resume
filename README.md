# HTML Resume

This is a simple resume webpage built using only **HTML**.

I created this project as part of the **Web Dev Cohort 2026** assignment to practice writing clean, structured, and semantic HTML without using CSS or JavaScript.
The goal was to focus purely on HTML elements such as headings, lists, tables, and sections.

## Purpose of the Project

* Practice semantic HTML
* Improve document structure and readability
* Build a resume layout without relying on styling or scripts
* Understand how raw HTML content is rendered by the browser

This project intentionally avoids CSS and JavaScript to keep the focus on fundamentals.
